🚀 YouTube Studio Monitor 사용법

=== 1. 프로그램 실행 ===

Windows:
  YouTubeStudioMonitor.exe 더블클릭

macOS:
  YouTubeStudioMonitor.app 더블클릭
  또는 터미널에서: ./YouTubeStudioMonitor

Linux:
  터미널에서: ./YouTubeStudioMonitor

=== 2. 수집 과정 ===

1) Chrome 자동 실행
   - 프로그램이 Chrome을 자동으로 열어줍니다

2) 로그인
   - 열린 Chrome에서 원하는 Google 계정으로 로그인
   - YouTube Studio (https://studio.youtube.com) 로 이동
   - 수집할 채널을 선택

3) 준비 완료
   - 비디오 목록이 보이는 화면까지 이동
   - 터미널에서 엔터 키 입력

4) 자동 수집
   - 프로그램이 자동으로 모든 데이터 수집
   - 5-10분 정도 소요됩니다

5) 결과 확인
   - youtube_analytics_data_날짜시간.xlsx 파일 생성
   - 썸네일 이미지와 모든 데이터가 포함됨

=== 3. 주의사항 ===

• Chrome이 미리 설치되어 있어야 함
• 안정적인 인터넷 연결 필요  
• 수집 중에는 Chrome 창을 닫지 마세요
• 다른 Chrome 창은 닫아주세요

=== 4. 문제 해결 ===

Chrome을 찾을 수 없다고 나오면:
  → Chrome 재설치 후 재시도

프로그램이 멈춘 것 같으면:
  → 5-10분 정도 기다려보세요
  → Chrome에서 새로고침 후 다시 시도

=== 5. 생성되는 파일들 ===

• youtube_videos_list_날짜시간.json : 비디오 목록 데이터
• youtube_analytics_data_날짜시간.json : 애널리틱스 데이터  
• youtube_analytics_data_날짜시간.xlsx : 최종 엑셀 파일 (가장 중요!)

엑셀 파일에는 다음이 포함됩니다:
- 비디오 썸네일 이미지
- 제목, 조회수, 노출수, 클릭률
- 시청시간, 평균 시청시간, 평균 조회율
- 비디오 URL 링크
- 게시일, 길이, 공개상태 등

🎉 완료!
